ilo.qa Installation
-------------------

To install ilo.qa using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ilo.qa`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ilo.qa

* Re-run buildout, e.g. with:

    $ ./bin/buildout
